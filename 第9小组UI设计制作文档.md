## 一、设计概述

### 1.1 设计目标

「简厨」是一款面向厨房新手及选择困难症用户的菜谱应用。UI 设计遵循以下核心原则：

- **简洁直观**：降低用户认知负担，首页即展示核心内容，无需多层级导航即可浏览菜谱。
- **一致性**：全局采用统一的橙色主题色（`#FF5C00` / `#D64D00`）、Poppins 字体族和 Material Design 组件规范。
- **内容优先**：使用大尺寸食物封面图吸引视觉焦点，文字信息层级清晰（标题 > 时间 > 正文）。
- **流畅动效**：采用 MotionLayout 实现侧边栏滑入动画，提升交互体验。

### 1.2 目标用户

| 用户类型 | 使用场景 | 设计考量 |
|---------|---------|---------|
| 厨房新手 | 搜索菜名，查看详细步骤 | 步骤说明清晰分行，配合图片 |
| 选择困难症患者 | 浏览推荐菜谱，获取灵感 | 首页横向卡片流展示，"SEE ALL" 引导浏览全部 |
| 在校学生 | 快速找到简单易做的菜品 | 菜谱按早/午/晚分类，提醒功能辅助时间管理 |

## 二、设计规范

### 2.1 色彩系统

| 角色 | 色值 | 用途 |
|------|------|------|
| 主色（Primary） | `#FF5C00` | 按钮、强调文字、导航栏背景、品牌标识 |
| 深主色（Primary Dark） | `#D64D00` | 侧边栏菜单项背景、状态栏 |
| 文字黑 | `#000000` | 正文标题、步骤正文 |
| 分隔灰 | `#C4C4C4`（25% 透明度） | 分隔线 |
| 白色 | `#FFFFFF` | 页面背景、卡片背景 |
| 辅助灰 | `#C4C4C4` | 搜索栏提示文字 |

### 2.2 字体规范

| 字体 | 文件 | 用途 |
|------|------|------|
| Poppins Bold | `poppins_bold.ttf` | 大标题（24sp/34sp）、按钮文字、导航菜单项 |
| Poppins Regular | `poppins_regular.ttf` | 正文内容、时间信息、说明文字 |

> 选用 Poppins 字体族的原因：几何无衬线风格现代感强，在中文环境中作为英文/数字展示字体与中文字体搭配和谐。

### 2.3 间距与布局

- **页面左右边距**：采用 Guideline 约束，左右各留 5.6% 屏幕宽度（`layout_constraintGuide_percent="0.056"` / `0.944`）。
- **卡片圆角**：16dp（顶部 App Bar）、15dp（设置页卡片）、8dp（菜谱封面卡片）。
- **卡片阴影**：10dp（首页 App Bar）、0dp + 1dp stroke（设置页）。
- **内容间距**：标题与内容间距 25-28dp，分隔线与下方内容间距 35-38dp。

### 2.4 图标系统

采用矢量图标（Vector Drawable），统一使用白色填充，尺寸规范：

| 图标 | 尺寸 | 用途 |
|------|------|------|
| `ic_baseline_menu_24` | 24dp | 首页汉堡菜单按钮 |
| `ic_baseline_arrow` | 24dp | 各子页面返回箭头（白色） |
| `ic_black_arrow` | 24dp | 关于页面返回箭头（黑色） |
| `ic_carbon_home` | 84dp | 侧边栏「主页」图标 |
| `ic_carbon_settings` | 84dp | 侧边栏「偏好」图标 |
| `ic_information` | 512vp | 侧边栏「关于」图标 |
| `app_logo` | 自适应 | 通知栏小图标 |

## 三、页面设计

### 3.1 主页（HomeFragment）

**布局结构：**

```
┌──────────────────────────────────┐
│  ┌──────────────────────────┐   │  ← 顶部 App Bar（圆角 CardView）
│  │ ☰  你今天想烹饪出什么美食？│   │     悬浮于内容之上，带 10dp 阴影
│  └──────────────────────────┘   │
│                                  │
│  ┌──────────────────────────┐   │
│  │ 我的米其林菜谱    SEE ALL │   │  ← 区域标题，右侧可点击查看全部
│  └──────────────────────────┘   │
│                                  │
│  ┌─────┐ ┌─────┐ ┌─────┐      │  ← 横向 RecyclerView（GridLayoutManager, spanCount=2）
│  │ img │ │ img │ │ img │      │     每张卡片 140dp 宽，含封面图+菜名+烹饪时间
│  │菜名 │ │菜名 │ │菜名 │      │
│  │时间 │ │时间 │ │时间 │      │
│  └─────┘ └─────┘ └─────┘      │
│                                  │
│  ─────────────────────────────  │  ← 半透明分隔线
│                                  │
│    Made with ♥ by Group A9      │  ← 底部签名
│                                  │
│         ┌──────┐                │  ← Loading 圆形进度条（网络加载时显示）
│         │  ⟳   │                │
│         └──────┘                │
└──────────────────────────────────┘
```

**设计要点：**
- App Bar 采用圆角悬浮设计（`cardCornerRadius="16dp"`, `cardElevation="10dp"`），营造层次感。
- 菜谱卡片采用横向滚动布局，引导用户快速浏览。
- 点击「SEE ALL」进入菜谱列表页（ListFragment）。
- 点击汉堡菜单图标打开左侧导航抽屉。

**设计要点：**

- 点击卡片进入菜谱详情页。

### 3.3 菜谱详情页（DetailFragment）

**布局结构：**

```
┌──────────────────────────────────┐
│  ←  简厨                         │  ← AppBarLayout + Toolbar
│──────────────────────────────────│
│  ┌──────────────────────────┐   │
│  │                          │   │  ← 食物封面图（CardView, 4:3 比例, 圆角 8dp）
│  │       食物封面图          │   │     使用 Coil 加载网络图片
│  │                          │   │
│  └──────────────────────────┘   │
│                                  │
│          菜品名称                 │  ← 大标题（Headline24dp，主题色）
│    准备时间: X Mins  制作时间: X   │  ← 水平排列的时间信息
│  ─────────────────────────────  │
│  原料                           │
│  食材1  数量                    │  ← 原料列表（支持 \n\n 换行分隔）
│  食材2  数量                    │
│  ─────────────────────────────  │
│  烹饪步骤                       │
│  （1）步骤一                    │  ← 步骤指南（支持 \n\n 换行分隔）
│  （2）步骤二                    │
│  （3）步骤三                    │
│  ─────────────────────────────  │
│  ┌──────────────────────────┐   │
│  │       分享菜谱             │   │  ← MaterialButton（圆角 15dp）
│  └──────────────────────────┘   │
└──────────────────────────────────┘
```

**设计要点：**
- 信息按「封面 → 标题 → 时间 → 原料 → 步骤 → 分享」的逻辑顺序排列。
- 分隔线使用半透明灰色（`#C4C4C4`, alpha=0.25），避免视觉割裂感。
- 底部「分享菜谱」按钮调用系统分享功能，将菜品信息以文本形式分享。

│  └──────────────────────────┘   │
│  ┌──────────────────────────┐   │
│  │ 午餐              [开关] │   │

### 3.6 侧边导航抽屉（Navigation Drawer）

```
┌──────────┐
│          │
│   🏠     │  ← 主页（白色图标 84dp）
│  主页    │
│──────────│
│   ⚙️     │  ← 偏好
│  偏好    │
│──────────│
│   ℹ️     │  ← 关于
│  关于    │
│          │
└──────────┘
```

**交互实现：**
- 基于 `DrawerLayout` + 自定义 `DrawerContent`（MotionLayout 子类）实现。
- MotionLayout 监听 Drawer 的 `onDrawerSlide` 回调，将 `slideOffset` 映射为 MotionLayout 的 `progress`，实现内容区域随抽屉滑动联动偏移 175dp。
- 抽屉背景透明（scrimColor 设为 transparent），通过 MotionLayout 动画替代传统的半透明遮罩。

### 3.7 加载状态（LoadingView）

- 居中显示 100×100dp 的 `ProgressBar`，初始 `visibility="gone"`。
- 网络请求开始时显示（`VISIBLE`），请求完成或出错后隐藏（`GONE`）。

## 七、主题适配

应用支持日间/夜间模式主题切换（通过 `AppCompatDelegate`），当前默认锁定为日间模式（`MODE_NIGHT_NO`）。

### 主题定义

| 主题 | 继承自 | 特点 |
|------|--------|------|
| `Theme.COOK.it`（日间） | `Theme.MaterialComponents.DayNight.NoActionBar` | 橙色主色，无默认 ActionBar |
| `Theme.COOK.it`（夜间） | `Theme.MaterialComponents.DayNight.DarkActionBar` | 橙色主色，暗色背景 |

### 自定义文字样式

| 样式名 | 字号 | 用途 |
|--------|------|------|
| `Headline34dp` | 34sp | 应用名称（关于页） |
| `Headline24dp` | 24sp | 页面标题、区域标题 |
